    Como foi feito ?

    Foi ministrada uma aula teórica sobre o github, explicando para os aluno o que era aplataforma, para que ela servia e a importancia dela para os desenvolvedores.

    Depois foi ministrada uma aula pratica ensinando os alunos a criarem uma conta no GitHub, criarem um repositório e atravéz desse repositório gerarem um link para assim o site ficar hospedado.

    Metodologia

    As metodologias utilizada para essa aula foram as:

    • 🖊 Resolução de problemas
    • 🖊 sala de aula invertida

    A metodologia de resolução de problemas foi utilizada pois os alunos tinham o site apenas no computador, não tinham ele hospdado, então conseguimos solucionar esse problema utilizando a plataforma do GitHub

    Já a metodologia da sala de aula invertida foi utilizada pois após a aula teórica os alunos deveriam pesquisar sobre o Github eter uma leve noção sobre o que era e qual a sua ideia central, assim trazer esse conhecimento adquirido em sala juntamente com a pesquisa realisada em casa para a proxima aula.
